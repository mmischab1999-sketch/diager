\######## DIAGER


Diager - powershell модуль, позволяющий сократить длинну команд диагностики и получения статусов процессов или служб до нескольких букв.



\---=== УСТАНОВКА ===---

В консоль Microsoft Powershell версии 5.1+:



Install-Module -Name Diager -force



\---=ПОСЛЕДНИЕ ОБНОВЛЕНИЯ=---

1.0.1 alpha - функция Get-Batteryreport , баннер при вводе первой команды после установки 

1.0.2 alpha - незначительные исправления в коде 



Функционал модуля (альфа версия 1.0.2) представляет собой 3 команды:
Get-DiskStress (псевдоним gds) - выводит состояние свободного места дисков (число, надобность в отчистке), состояние здоровья, надобность дефрагментации. (параметры -free -heath -defrag выводят все по отдельности соответственно)

Get-ServStatus (псевдоним gnss) - выводит статусы важных служб (FireWall, IPsec protocol, Microsoft Defender, Security center, Windows update, Time service, Audio Service).

Get-Batteryreport (псевдоним gbr) - только для ноутбуков, выводит состояние и информацию о батарее ноутбука

\---=== ROADMAP ===---
Get-Lag - позволит посмотреть температуру компонентов, загруженность ЦП, проблемы видеокарты, процессора - то есть возможные проблемы зависаний, подсвечивая проблемные места красным цветом

Get-Internet - позволит проверить все службы и процессы, связанные с интернетом - что может помочь выяснить, на каком именно отрезки пути от вашего компьютера к сайту возникают проблемы.

Команды исправления: отчистка временных файлов, старых обновлений, закрытие процессов, нагружающих систему, запуск необходимых служб. - то есть возможность исправления проблем, найденных в ходе диагностики.
